Shaming meeting room hogs

2016-10-01 1 min read
  • code

    One of the first things felt by a fast growing company is the lack of meeting space. The first few weeks at a new office it’s wonderful to know you can find a room whenever you need it. Yet after a few months and a bunch of extra people you realize you have to book meetings days in advance. And what makes this worse is seeing more than one room booked for the same meeting.

    After seeing this happening I decided to do something about it and wrote a quick script to pull the meeting calendar for every room from Google Calendar and then flag the ones having the same start time, end time, and creator. This isn’t foolproof since it won’t identify cases where someone books multiple rooms for the same time with different times but it’s a solid start and already caught a few cases. The code is up on GitHub so feel free to take a look and provide suggestions.
